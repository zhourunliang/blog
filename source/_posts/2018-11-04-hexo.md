## 使用Hexo主题[Yilia](https://github.com/litten/hexo-theme-yilia)

## 常用命令
- 安装
```bash
npm install -g hexo-cli
```

- 清空缓存
```bash
hexo clean
```

- 生成缓存
```bash
hexo generate
```

- 启动服务
```bash
hexo server
```

- 部署
```bash
hexo deploy
```
