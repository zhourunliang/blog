# git常用命令
- 从远程克隆一份到本地可以通过
```bash
git clone https://github.com/zhourunliang/zhourunliang.github.io
```

- 本地库关联远程库
```bash
git remote add origin https://github.com/zhourunliang/zhourunliang.github.io
```

- 推送master分支的所有内容
```bash
git push -u origin master
```
